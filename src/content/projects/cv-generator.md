This very site! A React-based CV generator with PDF export, multiple templates, and now an interactive globe.

## Features

- **Multiple Templates**: Support for different CV styles and themes
- **PDF Export**: Generate professional PDFs using Typst
- **Interactive Globe**: 3D visualization of work locations and projects
- **Responsive Design**: Works beautifully on all devices
- **YAML-based Content**: Easy content management

## Technology Stack

Built with modern web technologies including React, TypeScript, D3.js for visualizations, and Vite for blazing-fast development.

## Architecture

The application uses a template registry system allowing for easy customization and theming, with separate web and PDF rendering pipelines.
