Collection of open-source tools and utilities for humanitarian data collection and analysis.

## Purpose

These tools address specific challenges faced by humanitarian workers in the field, from data cleaning to visualization and reporting.

## Tools Included

- Data validation utilities
- Survey form converters
- Analysis scripts and notebooks
- Reporting templates
- Field data collection helpers

## Community

Open-source contributions welcome from the humanitarian tech community. Designed to be practical, lightweight, and field-tested.
